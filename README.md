## To-do List App

Simple, boring to do list app that has the basics.

### Features
- Add, search, edit, deleted tasks
- Theme toggle
- Local storage

### Built with 

React, 
Tailwind CSS,
Vite

### What You Can Do

- Add, edit, delete and update tasks
- Search tasks by keywords
- Check completed tasks
